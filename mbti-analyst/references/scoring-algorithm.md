# 梯度分层与类型推断规则

## 1. 梯度分层算法

### 1.1 分层阈值
- Δ ≤ 3 → 视为并置（同一层级内的功能强度无实际差异）
- Δ 3–5 → 视为近似（同一层级内轻微差异）
- Δ 5–8 → 层级内显著差（可能位于同一层级的上下边界）
- Δ > 8 → 层级断层（自然层级边界）

### 1.2 分层步骤
1. 将 8 项功能按分数从高到低排序
2. 计算相邻功能之间的 Δ
3. 在 Δ > 8 处切分层级边界（优先）
4. 若首尾功能之间无 >8 的断层，则在最大 Δ 处切分
5. 命名 T1（优势层）、T2（辅助层）、T3（过渡带）、T4（劣势层）

### 1.3 结构性发现模板
分层层级后，必须报告以下结构性发现：
- **T1 内部分布**：主导功能是单一还是双高？差距多大？
- **T1→T2 断层幅度**：显著断层（>8）提示明确的意识-无意识边界
- **T4 最弱功能**：全局最低功能及其与次低功能的差距
- **判断-感知轴平衡**：判断功能（Fi+Fe+Ti+Te）与感知功能（Ni+Ne+Se+Si）的总分对比
- **内倾-外倾平衡**：内倾功能（Fi+Ni+Ti+Si）与外倾功能（Fe+Ne+Te+Se）的总分对比
- **功能轴内的分裂**（如 Fi-Fe 差距 >15 需特别标注）

### 2.3 类型推断算法

1. **主导-劣势轴锁定**：找到最高分功能（候选主导）及其同轴相反态度功能。若相反态度功能为全局最低或次低 → 该轴为主轴，强证据。
2. **辅助-第三确认**：在剩余功能中，找到与主导同态度（内倾/外倾）相反的功能——该功能应为辅助；其同轴相反态度功能为第三。
3. **逐位置评分**：将 8 项实测分数与候选类型的 8 个位置期望分数范围对比，计算匹配度（✓ 吻合 / ≈ 可接受 / ✗ 偏离）。
4. **影子功能检验**：检查 5-8 位功能是否符合期望的低位。异常升高的影子功能需要特别解释。
5. **奥卡姆剃刀排序**：所需额外假设越少的类型越优先。每个异常升高的影子功能 = 一个额外假设。
6. **输出**：排名前两位的类型 + 概率百分比 + 匹配论据 + 偏离分析。

### 2.4 常见偏离模式

| 偏离模式 | 解释 | 示例 |
|----------|------|------|
| Ti≈Ne 并置 | ENTP 的辅助 Ti 反超主导 Ne（在学术/技术环境常见），或 INTP 的 Ti-Ne 高度协同 | 本 session 样本二 |
| Fi-Critic 异常升高 | 价值观冲突经历；大量阅读/内省；身份探索 | 本 session 样本一（INFJ 的 Fi 68 > Ni 66） |
| Demon Ni 异常升高 | 极端压力迫使 Ne→Ni 防御性收缩 | 本 session 样本二（若为 INTP，Ni 55 在辅助层） |
| Fe 极低（<30） | 若为 INTP 劣势——结构性薄弱；若为 ENTP 第三——发育停滞 | Fe=29 需两种框架对比分析 |
| Si 低于影子功能 | 第三功能发育滞后——年龄或环境的自然结果 | 本 session 样本二的 INTP 假设弱点 |

### 2.5 类型不确定性的诚实表达

- 必须在最终诊断中承认不确定性和替代假设
- 使用概率（如 55-65%）而非确定陈述
- 解释为何某一类型被优先排序（奥卡姆剃刀原理）
- 当与用户的实际体验冲突时，以用户反馈为准并调整概率
