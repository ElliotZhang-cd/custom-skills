## 1. 梯度分层算法

### 1.1 分层阈值（百分制）

- Δ ≤ 3 → 并置（同一层级内的功能强度无实际差异）
- Δ 3–5 → 近似（同一层级内轻微差异）
- Δ 5–8 → 层级内显著差（可能位于同一层级的上下边界）
- Δ > 8 → 层级断层（自然层级边界）

### 1.2 分层步骤

1. 将 8 项功能按分数从高到低排序
2. 计算相邻功能之间的 Δ
3. **在所有 Δ>8 处切分层级边界**
4. 边界数 = 0（全序列无任何 Δ>8 断层）→ **声明"无显著分层"**：不强行切层，报告按整体功能群描述；所有类型/依恋推断降权处理（主观置信度整体下调一档），并建议来访者重测
5. 切出层数 >4 → 合并"间距最小的相邻层"，直至 ≤4 层
6. **切出几层就命名几层**（T1、T2、T3……），禁止硬凑四层；若全局最低功能与其他功能之间无断层，明确写"本次数据没有独立的最低层级"

### 1.3 结构性发现模板

分层后，必须逐项报告以下结构性发现：
- **T1 内部分布**：主导功能是单一还是双高并置？差距多大？
- **T1→T2 断层幅度**：显著断层（>8）提示清晰的"自然工具 vs 用力工具"边界
- **全局最低功能**：及其与次低功能的差距
- **判断-感知轴平衡**：判断功能（Fi+Fe+Ti+Te）与感知功能（Ni+Ne+Se+Si）的总分对比
- **内倾-外倾平衡**：内倾功能（Fi+Ni+Ti+Si）与外倾功能（Fe+Ne+Te+Se）的总分对比
- **功能轴内的分裂**：Fi-Fe 差距 >15 需特别标注；Ti-Te、Ni-Ne、Si-Se 差距 >15 同理

### 2.2 Beebe 八功能完整位置

> 说明：Beebe 原型模型是荣格学派内部的理论建构，**无实证常模数据**。下表"期望分数范围"仅为理论预期，用于结构比对，不作为测量标准。

| 位置 | 名称 | 功能 | 理论预期 | 行为特征 |
|------|------|------|------------|----------|
| 1st | Hero/主导 | 同 MBTI 主导 | T1（最高） | 自然的、自动的、自我认同的 |
| 2nd | Parent/辅助 | 同 MBTI 辅助 | T1-T2（次高） | 支持主导的工具性功能 |
| 3rd | Child/第三 | 同 MBTI 第三 | T2-T3（中等） | 不稳定、需要滋养、脆弱 |
| 4th | Anima/劣势 | 同 MBTI 劣势 | 最低/次低 | 被投射的、防御性的 |
| 5th | Opposing/第五 | 与主导同轴相反态度 | T2-T3 | 对抗性的、在压力下被激活 |
| 6th | Critic/第六 | 与辅助同轴相反态度 | T3-T4 | 批判性的、审查性的 |
| 7th | Trickster/第七 | 与第三同轴相反态度 | 低位 | 制造双重束缚、最弱的意识控制 |
| 8th | Demon/第八 | 与劣势同轴相反态度 | 极低 | 最具破坏性、最后被整合 |

### 2.3 类型推断算法

1. **主导-劣势轴锁定**：找到最高分功能（候选主导）及其同轴相反态度功能。若相反态度功能为全局最低或次低 → 该轴为主轴，强证据。
2. **辅助-第三确认**：在剩余功能中，找到与主导同态度（内倾/外倾）相反的功能——该功能应为辅助；其同轴相反态度功能为第三。
3. **逐位置评分**：将 8 项实测分数与候选类型的 8 个位置理论预期对比，计算匹配度（✓ 吻合 / ≈ 可接受 / ✗ 偏离）。
4. **影子功能检验**：检查 5-8 位功能是否符合理论预期的低位。异常升高的影子功能需要特别解释。
5. **奥卡姆剃刀排序**：所需额外假设越少的类型越优先。每个异常升高的影子功能 = 一个额外假设。
6. **输出**：排名前两位的类型 + 主观置信度（见 §2.5）+ 匹配论据 + 偏离分析。

### 2.4 常见偏离模式

| 偏离模式 | 解释 | 示例 |
|----------|------|------|
| Ti≈Ne 并置 | ENTP 的辅助 Ti 反超主导 Ne（在学术/技术环境常见），或 INTP 的 Ti-Ne 高度协同 | `examples/cognitive_analysis_2.html` |
| Fi-Critic 异常升高 | 价值观冲突经历；大量阅读/内省；身份探索 | `examples/cognitive_analysis.html`（INFJ 的 Fi 68 > Ni 66） |
| Demon Ni 异常升高 | 极端压力迫使 Ne→Ni 防御性收缩 | `examples/cognitive_analysis_2.html`（INTP 假设下 Ni 55 偏高） |
| Fe 明显靠后（<35） | 若为 INTP 劣势——结构性薄弱；若为 ENTP 第三——发育停滞 | 需在两种框架下对比分析 |
| Si 低于影子功能 | 第三功能发育滞后——年龄或环境的自然结果 | `examples/cognitive_analysis_2.html` 的 INTP 假设弱点 |

### 2.5 类型不确定性的诚实表达

- 使用"**主观置信度**"（如 55–65%），禁止单独使用"概率"一词；首次出现必须附固定说明：
  > "置信度是分析者基于功能结构比对的主观估计，不是统计概率。它表示'在现有数据下，这个判断有多大把握'，而不是'你有百分之多少的可能性是这个类型'。"
- 必须输出前两名候选类型 + 各自的支持论据与解释难点
- 解释为何某一类型被优先排序（奥卡姆剃刀原理）
- 当与来访者的实际体验冲突时，以来访者反馈为准并调整置信度

### 2.6 阴影面（5-8 位）的通俗输出规则

分析层使用完整 Beebe 位置（§2.2），来访者正文**不出现原型名称**（"评论家/骗子/恶魔"禁用，仅限可选阅读附录并需引号+解释）。

画像章"压力下的你"小节的推导与写作规则：
1. **素材来源**：候选类型第 5-8 位功能 + 实测中的异常项（异常升高的影子功能、全局最低功能）
2. **输出内容**：压力下最可能冒出来的 1-2 个模式，全部 🔶 理论推断
3. **措辞公式**：压力情境 → 可能的表现 → "比如"具体场景 → 一句中性的应对提示（不教导、不开处方）
4. **禁忌**：禁止写成"你压力大时会变成 XX 型"（类型互换无实证依据）；禁止灾难化；禁止原型名称

常见映射（通俗版）：
| 实测结构 | 压力下的可能模式（🔶） |
|----------|----------------------|
| 第 6 位（"评论家"位）功能异常升高 | 内心的挑剔声音变响——对自己、也可能对亲近的人，标准突然变得苛刻 |
| 第 4 位（劣势）+ 第 8 位双双垫底 | 平时被忽略的通道在压力下"宕机式"反扑：如 Se/Si 双低者可能突然身体垮掉、或对琐碎细节产生反常的执念 |
| 第 5 位（"对立"位）偏高 | 压力下可能出现与平时相反的态度：平时收敛的人突然抬杠、唱反调 |


## 3. 输入质量前置检查（在分层之前执行）

1. **完整性**：必须 8 项齐全、每项 0–100；缺项/超界 → 退回要求重发
2. **十分制护栏**：若 8 项全部 ≤10 → 不静默处理，反问确认是否为十分制量表
3. **扁平剖面检测**：全距（最高分−最低分）<10 → "本次测试区分度不足"，建议重测；所有推断降权
4. **作答质量检查**（纯数据模式识别，不是诊断）：
   - 全部 ≥85 或全部 ≤30，且全距 <10 → 可能存在极端应答或敷衍作答
   - 出现以上模式 → 在向咨询师汇报时警告，并在报告中弱化结论强度
