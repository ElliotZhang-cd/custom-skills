# 依恋类型推断的认知功能映射规则

## 1. 核心映射逻辑

### 1.1 依恋回避维度 ← 认知功能

**回避的操作性定义**：对情感亲密的不适、偏好情感自给自足、不倾向于依赖他人获取情绪调节。

| 功能 | 与回避的关系 | 机制 |
|------|------------|------|
| **Fe ↓** | 回避 ↑ | Fe 是实现情感亲密的主要认知工具。Fe 越低，个体越难以自然读取和回应他人的情感信号，越难建立情感亲密。 |
| **Fi ↑ + Fe ↓** | 回避 ↑ | 高 Fi 提供内在情感自给自足——"我能自己处理情绪"——降低对外部情感支持的需求感知。但这里的回避是策略性的（我能，所以我不想依赖），而非恐惧性的。 |
| **Ti ↑** | 回避 ↑（间接）| 高 Ti 倾向于理智化防御——将情感体验重新框定为可分析的问题。这是疏离型依恋的核心防御机制之一。 |

### 1.2 依恋焦虑维度 ← 认知功能

**焦虑的操作性定义**：对伴侣情感可用性的过度警觉、对被抛弃的强烈恐惧、需要频繁的情感确认。

| 功能 | 与焦虑的关系 | 机制 |
|------|------------|------|
| **Fi ↑ + Ni ↑** | 焦虑 ↑（特定情境）| Fi-Ni 内循环可将微小的关系信号放大为完整的灾难叙事。Fi 将伴侣行为编码为价值信号（"Ta 这样做 = Ta 不够在乎我"），Ni 将此信号编织为连贯的意义模式。 |
| **Si ↓** | 焦虑 ↑↑ | Si 是"内在安全基地"的认知基础——将过往被安抚的经验内化为可自主调用的身体/情感记忆。Si 越低，个体越依赖伴侣的实时在场来调节情绪。 |
| **Fe ↑** | 焦虑 ↓（部分缓冲）| 较高的 Fe 可以通过读取伴侣的情绪状态获得"客观"的关系安全信号，部分减轻焦虑。但 Fe 不能替代 Si 的自主安抚功能。 |
| **Se ↓** | 焦虑 ↑（间接）| Se 低限制了个体通过身体活动和感官投入来分散注意力和调节情绪的能力。 |

## 2. 依恋类型的认知功能签名

### 2.1 安全型
- Fe 与 Fi 相对均衡（Δ < 10）
- Si 处于中等水平（> 45）
- 感知功能无集体塌陷

### 2.2 焦虑-痴迷型（Anxious-Preoccupied）
- **Si < 40**（最关键的标志——缺乏内在安全基地）
- Fi 偏高（> 55）——将伴侣纳入核心价值圈，投注极深
- Ni 偏高 + Ne 偏低 → 认知聚焦于关系的单一叙事
- Fe 中等——能够读取伴侣情绪，但此读取服务于焦虑系统的过度警觉
- Bartholomew 定位：消极自我模型 + 积极他人模型

### 2.3 疏离-回避型（Dismissive-Avoidant）
- **Fe < 35**（最关键的标志——情感表达的结构性薄弱）
- Fi 中等偏高（> 50）——情感自给自足
- Ti 偏高——理智化防御
- Si 低至中低
- Bartholomew 定位：积极自我模型 + 消极他人模型

### 2.4 恐惧-回避型（Fearful-Avoidant）
- **Fe < 35 + Si < 40 + Fi 高**（罕见但可能）
- 同时呈现焦虑型和疏离型的认知功能签名
- 对亲密既渴望又恐惧——Fi 渴望深度连接，但 Fe 无能实现，Si 无能内化
- Bartholomew 定位：消极自我模型 + 消极他人模型

## 3. 关系领域特异性推断

### 3.1 理论基础
Fraley et al. (2011) 和 Overall et al. (2003) 证实：个体可在不同关系领域持有不同的依恋表征，不同领域间的依恋相关仅 r≈0.3-0.5。

### 3.2 伴侣关系 vs 父母关系的分化机制

| 关系领域 | 典型依恋特征 | 认知功能基础 |
|----------|------------|-------------|
| **浪漫伴侣** | 可能是焦虑型（如果 Fi 极高 + Si 极低）| Fi 的排他性信任——"伴侣是我选择的人，代表我真正想要的关系"。对通过价值观审核的伴侣投入全部情感资源。 |
| **父母/亲人** | 可能是疏离型（如果 Fe 低 + Si 低）| Fi 的价值独立性——"我不需要你们的认可来确认我的价值"。对早期照护者的情感可得性持有防御性低期待。 |

### 3.3 分化推断规则
1. 如果 Si < 40：伴侣关系中的焦虑概率显著升高（无法内化安全经验 → 依赖伴侣实时安抚）
2. 如果 Fe < 35 且 Si < 40：可能在所有关系中呈现不安全依恋，但分化模式取决于 Fi 的"遴选"——Fi 将不同的情感投注分配到不同的关系对象
3. 如果用户提供了具体依恋表现（如"伴侣焦虑型/父母疏离型"），以用户指定为准，功能分析用于解释而非推断

### 3.4 阈值边界敏感规则（强制）

- 任何分数落在阈值 **±3** 边界上（如 Si 在 37–43 之间、Fe 在 32–38 之间），基于该阈值的所有结论必须标注：
  > "此结论对分数波动敏感——该分数正处于判断边界上，重测时轻微变化即可能改变此结论。"
- 边界敏感的结论同时列入生成时向咨询师汇报的"咨询师备注"清单
- 禁止在边界状态下输出确定性的依恋类别名称；只能说"倾向于……的方向"

## 4. 推断的边界与标注

### 4.1 必须标注的内容
- 所有依恋推断必须标注"此为基于功能分数的理论推断，非临床诊断"
- 若用户未提供关系特异性信息，标注"以下推断未经验证，建议结合 ECR-R 或 AAI 进行正式评估"
- 在双人分析中，若一方的依恋类型为推断而非用户报告，标注"推测"
- 证据分级（见 writing-style.md §5）：功能→依恋的映射整体为 ⚪ 探索性假设；依恋的"领域特异性"（不同关系中依恋模式可不同）为 ✅ 研究支持（Fraley et al., 2011; Overall et al., 2003）
- 来访者报告正文中，依恋章末尾必须照录固定谦卑段落（writing-style.md §7.1）

### 4.2 不能做的推断
- 不能直接推断童年经历的具体事件（可推断"成长环境的特征"，不可推断"被父母虐待"等具体事件）
- 不能诊断人格障碍或临床级别的依恋障碍
- 不能将依恋推断呈现为确定性结论
