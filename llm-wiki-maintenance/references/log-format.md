# 日志格式规范

## 单条格式

```
## [YYYY-MM-DD] type | 标题
- 摘要：一句话概述
- 创建页面：`wiki/...`（如有）
- 更新页面：`...`（如有）
- 标签：#tag1 #tag2
```

## 操作类型

`ingest` | `query` | `lint`

## 归档规则

log.md 超过 500 行时：
1. 最旧条目移至 `raw/logs/archive-YYYY-MM.md`
2. 主文件保留最近 30 条
3. 归档后在 index.md 原始资料表添加归档文件条目
