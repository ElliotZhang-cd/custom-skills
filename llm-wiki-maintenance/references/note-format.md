# 页面格式（当前约定）

> 这些是起点，不是终点。与用户共同迭代中。

## Frontmatter（当前约定）

```yaml
---
type: concept | entity | synthesis
tags: []
created: YYYY-MM-DD
sources: []     # 与正文「来源」段对应
---
```

## 正文结构（当前约定）

### concept（概念定义）

常见结构：
- `# 标题`
- `> 费曼摘要`（引用块，类比解释）
- `## 要点`（3-6 条，可选分组）
- `## 相关`（≥2 条 wikilink）
- `## 来源`（raw/ 或 URL）

### entity（实体简介）

常见结构：
- `# 实体名称`
- `> 一句话定义`
- `## 要点`（3-6 条，扁平）
- `## 相关`（≥2 条）
- `## 来源`

### synthesis（综合论证）

常见结构：
- `# 标题`
- `> 费曼摘要`
- `## 要点`（按原文逻辑链分组，每组 2-5 条）
- `## 相关`
- `## 来源`

> 💡 分组建议：按原文的论证结构（如"前提→机制→证据→瓶颈"）比按主题分类更容易让读者跟随逻辑。

## 文件命名

kebab-case 英文，专有名词保留大小写。
