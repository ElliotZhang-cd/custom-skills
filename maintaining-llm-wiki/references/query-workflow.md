# Query 查询流程

## 步骤

1. **读 index.md** → grep 定位相关页面
2. **精读页面** → 深入阅读相关 wiki 页面
3. **合成回答** → 标注 `[[wikilink]]` 来源

## 发现处理

查询中发现矛盾/过时/断链 → 告知用户 → 确认后按 Lint 修复流程处理。

## 回存规则

- **多源综合/对比分析/新发现** → 简述摘要 → 用户确认 → 写入 wiki → 执行 Ingest 步骤 3 起的写入流程
- **简单事实查询** → 不回存
- **边界情况** → 快速确认
